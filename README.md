How to Install Check360 (Chrome Extension)

Follow the steps below to set up Check360 in your Chrome browser:

1.Download the latest Check360 ZIP file from this repository.

2.Extract (unzip) the downloaded file to any folder on your computer.

3.Open Chrome and go to: chrome://extensions/

4.Enable Developer Mode using the toggle in he top right corner.

5.Click Load Unpacked.

6.Select the extracted Check360 folder.

7.Once loaded, click Update (optional but recommended).
